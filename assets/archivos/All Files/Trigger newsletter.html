Bases de Datos

Este curso introduce a los participantes a los conceptos básicos de bases de datos relacionales. El curso enseña terminología de base de datos relacional, así como conceptos de modelado de datos, creación de diagramas de relación de entidad (ERD) y mapeo de ERD.

Oracle SQL Developer Data Modeler se utiliza para construir ERD y The Structured Query Language (SQL) se usa para interactuar con un relacional base de datos y manipular la fecha dentro de la base de datos.

Programación en Java

Este curso involucra a estudiantes con poca o ninguna experiencia en programación para crear programas Java.


A los participantes se les presentan los conceptos, la terminología y la sintaxis de la programación orientada a objetos, y los pasos necesarios para crear programas Java básicos utilizando Alice, entornos de desarrollo interactivo Greenfoot y Eclipse.

CCNA
